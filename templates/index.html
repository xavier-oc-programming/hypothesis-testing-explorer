The Basics

Hypothesis testing A method for deciding whether a pattern in your data is real or just random chance. You collect two groups of measurements and ask: are these actually different, or could the gap be noise?
Null hypothesis The default assumption that there is no difference between the groups. The test tries to find enough evidence to reject this. If p < α, you reject it — the difference is real.
Statistical significance A result is "statistically significant" when the p-value falls below your threshold (α). It means the result is unlikely to be a fluke — not that the difference is large or practically important.
Parametric vs non-parametric Parametric tests (t-tests) assume your data follows a normal distribution. Non-parametric tests (Mann-Whitney) make no such assumption — they work by ranking the data instead.
Normal distribution The classic bell curve — most values cluster around the middle, fewer at the extremes. Many natural measurements follow this shape. Required for t-tests to be reliable.
Statistical power The test's ability to detect a real difference when one exists. More data = more power. A test with low power might miss a real effect and incorrectly report no significant difference.
Mean The average. Add all values and divide by how many there are. The t-tests in this app compare the means of two groups.
Variance How spread out the values are around the mean. High variance = values are scattered widely. Low variance = values cluster tightly. Two groups can have the same mean but very different variances.
Standard deviation (std) The square root of the variance — a more intuitive measure of spread, in the same units as the data. About 68% of values in a normal distribution fall within one standard deviation of the mean.
Median The middle value when all values are sorted. Unlike the mean, it is not pulled by extreme outliers. The box in a box plot shows the median as a line in the middle.
IQR (interquartile range) The range of the middle 50% of values — from the 25th to the 75th percentile. Shown as the height of the box in a box plot. A wide box means high spread in the middle of the data.
Outlier A value far from the rest of the data. Shown as individual dots beyond the whiskers in a box plot. Outliers can distort means and inflate variance — one reason to check normality before running a t-test.
Distribution The shape of your data — how values are spread across the range. The KDE curve (smooth line) in the left panel of the plot shows your data's distribution. A symmetric bell shape is normal.
False positive (Type I error) Concluding there IS a difference when there actually isn't one. The α threshold (0.05) is exactly the rate of false positives you're willing to accept — 5 in every 100 tests on truly identical groups would appear significant by chance.
False negative (Type II error) Concluding there is NO difference when there actually is one. Usually caused by too little data (low n) or a very small real difference. Increasing sample size reduces this risk.

Results

p-value The probability of seeing a difference this large if there were truly no difference. Below 0.05 → the result is called statistically significant. The lower it is, the stronger the evidence.
α (alpha) Your significance threshold — the cut-off you set before running the test. Default is 0.05, meaning you accept a 5% chance of a false positive. If p < α, the result is significant.
Test statistic A single number summarising how different the two groups are, relative to the natural variation in the data. The further it is from zero, the more different the groups. The p-value is calculated from this number.
n The number of observations (rows) in that group. Larger n gives the test more power to detect real differences.
Engine Which tool ran the computation. pandas/scipy is used for datasets under 50,000 rows. PySpark + scipy is used above that — PySpark prepares the data at scale, scipy computes the test statistic.

Assumption Checks

Normality (Shapiro-Wilk) Tests whether the data follows a bell-curve shape. Parametric tests (t-tests) assume this. W is the test statistic — close to 1.0 means the data looks normal. The p-value here works in reverse: high p = normal, low p = not normal.
Equal variance (Levene's) Tests whether both groups have a similar spread of values. The standard independent t-test assumes they do. W here measures the difference in spread. Again, high p = equal variance (assumption holds), low p = unequal (use Welch's t-test instead).
Assumption check p-values These work the opposite way to the main test p-value. For the main test, low p = significant result. For assumption checks, low p = the assumption is violated. You want these to be high (above 0.05).
Recommendation If an assumption is violated, the app suggests a more appropriate test. For example, non-normal data → Mann-Whitney U. Unequal variances → Welch's t-test.
